# 🛒 Customer Purchase Behavior & Sales Forecasting
### End-to-End Retail Analytics | Python · SQL · Machine Learning

---

## 📌 Overview
A full end-to-end data analytics project analyzing **100,000+ real Brazilian 
e-commerce orders** (Olist dataset) to uncover customer behavior patterns and 
forecast monthly revenue.

**Role:** Data Analyst  
**Tools:** Python, Pandas, SQL (SQLite), Matplotlib, Seaborn, Scikit-learn  
**Dataset:** [Olist Brazilian E-Commerce — Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)  
**Status:** 🔄 In Progress

---

## ❓ Business Questions
1. Which product categories generate the most revenue?
2. How has monthly revenue trended over time?
3. Which states have the slowest delivery times?
4. Does faster delivery lead to higher review scores?
5. What are the peak shopping hours and days?
6. Can we forecast next month's revenue?

## 🚀 How to Run
1. Clone this repo: `git clone https://github.com/ErSKM/olist-ecommerce-analysis.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Download dataset from Kaggle and place CSVs in `/data/`
4. Open and run `notebooks/olist_analysis.ipynb`
